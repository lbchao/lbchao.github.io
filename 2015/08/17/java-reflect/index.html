介绍java反射的知识

获得Class对象

在java中获得Class对象通常有以下几种方式：
1使用Class.forName();
2调用类的class属性获得；
3调用对象的getClass()方法获得
1,2都是根据类来获取类的Class对象，2有以下几种好处：
代码更安全，程序在编译阶段检查访问的对象是否存在，
程序性能更好，无需调用方法，性能更好。

Class中获得信息

通过class对象，可以获得类的构造方法，指定方法，注解方法，字段，实现的接口，父类，class类信息，

cls.getDeclaredConstructors(); 获得不分权限的构造方法
cls.getDeclaredFields()； 获得不分权限的字段

java8新增的方法参数反射

java8在java.lang.reflect包下新增了一个Executable抽象基类，该对象代表可执行的类成员，改类派生了Constructor,Method两个子类。
Executable类提供了大量方法类获取修饰方法或构造器的注解信息；还提供方法判断该方法或构造器是否包含数量可变的形参，以及通过getModifoers()方法获取方法或者构造器的修饰符，除此之外，提供了方法来获取该方法或参数的形参个数及形参名。

使用反射生成操作最对象

通过反射生成对象有以下两种方式：
1使用class对象的newInstance方法来创建该class对象的对应类的实例，这种方式要求class对象对应类有默认的构造器，执行该方法实际上利用默认构造器来创建实例。
2先使用class对象获取指定的Constructor对象，再调用Constructor对象的newInstance方法来创建class对象对应的实例，通过这种方法可以选择特定的构造器来创建实例。

调用方法

当获得某个类的Class对象后，就可以通过该对象getMethods()或者getMethod()方法来获取全部指定的方法，
每个Method对象对应一个方法，获得该对象后，程序就可以通过该Method调用它对应的方法，即invoke（Object obj,Object args）,第一个参数是object执行方法的主调，后边的是传入方法的实参。
如果某个对象的方法是private，在调用之前，设置Method对象的Accessible为true,取消权限的检查。

访问成员变量值

通过Class对象的getFields()方法或者getField()方法可以获取该类所包括的全部成员变量或指定成员变量。
Field提供了如下两种方法来读取或者设置成员变量值。
getXxxx(Object object)：获得object对象的该成员变量值，此处的Xxxx对应8种基本类型，如果该成员变量的类型是引用类型，则取消get后面的Xxxx。
setXxxx(Object object，Xxx value):将object对象的该成员变量值设置成value.此处对应8种基本类型，此处对应8种基本类型，如果成员变量是引用类型，取消后边的setXxx()方法。
使用这两个方法可以随意的访问指定对象的所有成员变量，包括private修饰的成员变量。

操作数组

java.lang.reflect包下还提供了一个Array类，程序通过这个类可以动态的操作数组，创建数组等。
