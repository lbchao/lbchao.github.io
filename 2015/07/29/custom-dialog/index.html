自定义对话框之一，继承Dialog类

首先创建一个类，继续android.app.Dialog类
1首先创建构造方法，设置主题，如下:

 public CustomSystemDialog(Context context) {
    super(context, R.style.dialog);
    initSetting(context);
}

dialog.xml主题文件

2然后新建一个方法，进行一些基本的配置 例如点击屏幕外，点击返回按钮的事件监听，自定义布局及自定义尺寸等。

3.可以创建 接口类，对对话框里的按钮事件进行监听，如下：

 public interface onClickDialogListener {
    void onClickOkListener();

    void onClickCancelListener();
}

public void setOnClickDialogListener(onClickDialogListener listener) {
    this.listener = listener;
}
